# PingPro Development Roadmap

## Current State (as of 2025-05-22)
- Static landing page live at https://wish210.github.io/pingpro/
- Responsive design, professional look
- No real products, no images, no backend, no payments
- Emoji placeholders for product cards
- Contact form doesn't submit anywhere

## Phase 1: Foundation (Week 1–2)

### 1.1 Site Upgrades
- [ ] Add SEO meta tags + Open Graph for social sharing
- [ ] Replace emoji product images with real blade photography
- [ ] Wire contact form to Formspree (free, 50 submissions/month)
- [ ] Add Google Analytics (GA4) for traffic tracking
- [ ] Create product page template (individual blade pages)

### 1.2 First Product Pages
- [ ] Yinhe V14 Pro ALC — flagship product page
- [ ] Sanwei FEXTRA 7 — mid-range bestseller page
- [ ] LOKI K5 Pro — budget performance page
- [ ] Each page: specs table, playing style match, comparison to expensive alternative

### 1.3 Blade Finder Tool
- [ ] Interactive quiz: skill level → playing style → budget → handle preference
- [ ] Recommends 1–3 blades from inventory
- [ ] Captures email for follow-up (lead generation even before inventory)
- [ ] Differentiator from every other store

### 1.4 Order Samples
- [ ] Order from AliExpress: Yinhe V14 Pro, Sanwei FEXTRA 7, LOKI K5 Pro, Sanwei 75 ALC
- [ ] Photograph upon arrival (natural light, multiple angles, with ruler for scale)
- [ ] Play-test each blade for 1 week minimum
- [ ] Write honest review with speed/spin/control ratings

## Phase 2: First Sales (Week 3–6)

### 2.1 Payment Integration
- [ ] Add Stripe Checkout (no backend needed — hosted checkout page)
- [ ] Create 3–5 product listings with "Buy Now" buttons
- [ ] Set up order notification emails
- [ ] Define shipping rates ($5 flat US, free over $75)

### 2.2 Fulfillment Setup
- **Initial method**: Buy 5–10 of each blade from AliExpress, ship from home
- [ ] Get USPS flat rate padded envelopes (blades fit perfectly)
- [ ] Create packaging solution (bubble wrap + cardboard stiffener)
- [ ] Set up shipping label printing (Pirate Ship or Shippo for discount rates)

### 2.3 Content Launch
- [ ] Write: "Best Chinese Table Tennis Blades 2025" (SEO article)
- [ ] Write: "Yinhe V14 Pro vs Butterfly Viscaria" (comparison)
- [ ] Write: "Complete Blade Buying Guide for Intermediate Players"
- [ ] Post to Reddit r/tabletennis (genuine, not spammy — share reviews)

### 2.4 Legal Basics
- [ ] Add Terms of Service page
- [ ] Add Return Policy page (30-day returns, unused condition)
- [ ] Add Privacy Policy (required for Stripe + analytics)
- [ ] Register business name if selling (sole proprietor is fine to start)

## Phase 3: Growth (Month 2–3)

### 3.1 Brand Relationships
- [ ] Contact Yinhe export department for distributor pricing
- [ ] Contact Sanwei for wholesale account
- [ ] Negotiate 30–40% below AliExpress price at 20–50 piece orders
- [ ] Request product images and marketing materials from brands

### 3.2 Expand Catalog
- [ ] Add 8–12 blade models (cover all categories: ALL, ALL+, OFF-, OFF, OFF+)
- [ ] Add rubber section (DHS Hurricane 3, Yinhe Moon, Sanwei T88)
- [ ] Add accessories (blade cases, edge tape, cleaning kits)
- [ ] Each with expert review and playing style recommendation

### 3.3 Customer Experience
- [ ] Add user reviews system (simple — even just testimonials)
- [ ] Email newsletter (new products, blade tips, promotions)
- [ ] Consider live chat for blade recommendations
- [ ] Loyalty program or referral discount

### 3.4 Marketing Scale
- [ ] Google Shopping ads for top 5 products
- [ ] TikTok content: "This $50 blade vs $185 Butterfly" (viral format)
- [ ] YouTube: unboxing + first impression + 1-week review format
- [ ] Build email list to 500+ subscribers

## Phase 4: Mature (Month 4–6)

### 4.1 Private Label
- [ ] Design "PingPro V1" blade specification (5W+2C, limba/ayous/carbon)
- [ ] Contact 2–3 OEM factories for quotes and samples
- [ ] Test samples extensively
- [ ] First production run: 50 pieces (~$500)
- [ ] Position as "house brand" at $65–$75 (insane value vs competition)

### 4.2 Platform Expansion
- [ ] List bestsellers on Amazon (FBA for Prime shipping)
- [ ] eBay store for broader reach
- [ ] Consider Shopify migration if GitHub Pages + Stripe becomes limiting

## Key Metrics to Track
- Blade Finder completion rate (engagement)
- Conversion rate by traffic source
- Average order value (AOV)
- Return rate (quality indicator)
- Customer acquisition cost (CAC)
- Email list growth rate
- Organic search impressions/clicks (SEO effectiveness)

## Critical Success Factors
1. **Blade quality**: Only sell blades you've personally tested and approve
2. **Authentic expertise**: Reviews must be genuine and data-driven
3. **US shipping speed**: This is the #1 differentiator vs AliExpress
4. **Niche focus**: Don't try to be Megaspin — be the Chinese-blade expert
5. **Content-first**: Build authority before scaling inventory
